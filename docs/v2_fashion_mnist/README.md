# Fashion-MNIST V2 Folder

This folder keeps Fashion-MNIST Version 2 documentation separate from MNIST Version 1.

Current V2 status: scope, data loader, training scripts, smoke tests, controlled runs, reliability check, and results summary are complete.

Use this folder for:

- V2 scope and planning
- V2 result summary
- V2 interpretation notes
- V2 limitations and portfolio-facing wording

Main files:

- `FASHION_MNIST_V2_SCOPE.md`
- `FASHION_MNIST_V2_RESULTS_SUMMARY.md`

Related V2 locations:

```text
scripts/v2_fashion_mnist/
tests/v2_fashion_mnist/
results/tables/v2_fashion_mnist/
src/kd_research/data/fashion_mnist.py
```

The `src/` file is shared package code, so it stays outside this folder. The V2 docs and scripts stay grouped here to avoid mixing Version 1 and Version 2 work.
