# Learning Notebooks

These notebooks are learning walkthroughs and personal study notes. They are not
the official source of controlled experiment results.

Official results are recorded in:

- `docs/CONTROLLED_EXPERIMENT_LOG.md`
- `results/tables/`
- the version-specific result summaries under `docs/`

## Notebook Guide

### V1 MNIST

- `v1_mnist/mnist_baseline_walkthrough.ipynb`  
  Step-by-step hard-label student baseline walkthrough.

- `v1_mnist/mnist_kd_walkthrough.ipynb`  
  Step-by-step teacher-student knowledge distillation walkthrough.

### V2 Fashion-MNIST

- `v2_fashion_mnist_data_loader_walkthrough.ipynb`  
  Data-loader walkthrough for the Fashion-MNIST V2 extension.

## How to Read These

Use the notebooks to understand the ideas and mechanics first. Use the `.py`
scripts, logs, CSV tables, and result summaries when citing controlled
experiment results.
